## Turtle Movement

| Command | Short | Description |
|---------|-------|-------------|
| `forward` | `fd` | Move forward by distance |
| `back` | `bk` | Move backward by distance |
| `right` | `rt` | Turn clockwise by degrees |
| `left` | `lt` | Turn counter-clockwise by degrees |
| `home` | | Return to center, heading north |
| `setxy` | | Set position to x, y coordinates |
| `setpos` | | Set position from `[x y]` list |
| `setx` | | Set x coordinate only |
| `sety` | | Set y coordinate only |
| `setheading` | `seth` | Set heading (0=north, 90=east) |
| `setc` | | Set position from complex number |
| `arc` | | Draw arc with angle and radius |

## Control Flow

| Command | Description | Example |
|---------|-------------|---------|
| `repeat` | Loop n times | `repeat 4 [fd 100 rt 90]` |
| `for` | Loop with counter | `for [i 1 10] [print :i]` |
| `while` | Loop while true | `while :x > 0 [make "x :x - 1]` |
| `until` | Loop until true | `until :x = 0 [make "x :x - 1]` |
| `forever` | Loop forever | `forever [fd 1 rt 1]` |
| `do.while` | Run then loop while | `do.while [fd 10] [:x > 0]` |
| `do.until` | Run then loop until | `do.until [fd 10] [:x = 0]` |
| `if` | Conditional | `if :x > 10 [fd 100]` |
| `ifelse` | If-then-else | `ifelse :x > 0 [fd :x] [bk :x]` |
| `test` | Set test flag | `test :x > 0` |
| `iftrue` / `ift` | Run if test was true | `iftrue [fd 100]` |
| `iffalse` / `iff` | Run if test was false | `iffalse [bk 100]` |
| `case` | Match value | `case :x [[1] [print "one] [2] [print "two]]` |
| `cond` | Multiple conditions | `cond [[:x > 0] [print "pos] [:x < 0] [print "neg]]` |
| `stop` | Exit procedure | `if :n = 0 [stop]` |
| `output` / `op` | Return value | `output :x * 2` |
| `repcount` | Current iteration | `repeat 5 [print repcount]` |

## List Operations

### Construction

| Command | Short | Description | Example |
|---------|-------|-------------|---------|
| `list` | | Create list | `list 1 2 3` → `[1 2 3]` |
| `sentence` | `se` | Create flat list | `se [1 2] [3 4]` → `[1 2 3 4]` |
| `fput` | | Prepend element | `fput 0 [1 2]` → `[0 1 2]` |
| `lput` | | Append element | `lput 3 [1 2]` → `[1 2 3]` |
| `combine` | | Concat or fput | |
| `range` / `iseq` | | Integer sequence | `range 1 5` → `[1 2 3 4 5]` |
| `rseq` | | Real sequence | `rseq 0 1 5` → `[0 0.25 0.5 0.75 1]` |
| `reverse` | | Reverse list | `reverse [1 2 3]` → `[3 2 1]` |

### Accessors

| Command | Short | Description | Example |
|---------|-------|-------------|---------|
| `first` | | First element | `first [1 2 3]` → `1` |
| `last` | | Last element | `last [1 2 3]` → `3` |
| `butfirst` | `bf` | All but first | `bf [1 2 3]` → `[2 3]` |
| `butlast` | `bl` | All but last | `bl [1 2 3]` → `[1 2]` |
| `item` | | Element at index | `item 2 [a b c]` → `b` |
| `count` | | Length | `count [1 2 3]` → `3` |
| `pick` | | Random element | `pick [1 2 3]` → random |
| `member` | | Tail from element | `member "b [a b c]` → `[b c]` |
| `memberp` | `member?` | Is member? | `member? "b [a b c]` → `true` |
| `firsts` | | First of each | `firsts [[1 2] [3 4]]` → `[1 3]` |
| `butfirsts` | `bfs` | Butfirst of each | |

## Word Operations

| Command | Description | Example |
|---------|-------------|---------|
| `word` | Concatenate words | `word "hel "lo` → `"hello` |
| `first` | First character | `first "hello` → `"h` |
| `last` | Last character | `last "hello` → `"o` |
| `butfirst` / `bf` | All but first char | `bf "hello` → `"ello` |
| `butlast` / `bl` | All but last char | `bl "hello` → `"hell` |
| `count` | Length | `count "hello` → `5` |
| `ascii` | ASCII value | `ascii "A` → `65` |
| `char` | Character from code | `char 65` → `"A` |
| `lowercase` | Convert to lower | `lowercase "HI` → `"hi` |
| `uppercase` | Convert to upper | `uppercase "hi` → `"HI` |

## Higher-Order Functions

| Command | Description | Example |
|---------|-------------|---------|
| `map` | Apply to each element | `map [? * 2] [1 2 3]` → `[2 4 6]` |
| `map.se` | Map and flatten | |
| `filter` | Keep matching elements | `filter [? > 2] [1 2 3 4]` → `[3 4]` |
| `find` | First matching element | `find [? > 2] [1 2 3 4]` → `3` |
| `reduce` | Fold with binary op | `reduce [?1 + ?2] [1 2 3 4]` → `10` |
| `foreach` | Apply for side effects | `foreach [print ?] [1 2 3]` |

In templates: `?` is current element, `?1` and `?2` for reduce.
